# NorthStar Databases and Analytics Coursework

This repository contains the coursework files for the Databases and Analytics assignment based on the NorthStar Urban Mobility and Logistics case study.

## Project Overview

The project investigates NorthStar's fragmented data environment and uses SQL in R, R analytics, Python data processing, MongoDB Atlas, and query optimisation to analyse operational and customer service issues.

## Repository Structure

- `data/` contains the CSV datasets used for the analysis.
- `notebooks/` contains the Google Colab notebooks used for SQL in R, R analytics, Python processing, MongoDB development, and query optimisation.
- `screenshots/` contains screenshots of code outputs, charts, MongoDB collections, CRUD operations, and query optimisation results.
- `report/` contains the final coursework report.

## Main Technologies Used

- SQL in R
- R analytics
- Python
- Pandas
- NumPy
- Matplotlib
- MongoDB Atlas
